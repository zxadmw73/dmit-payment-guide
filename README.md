## 几个常被问到的问题

**Q：DMIT 微信付款需要实名认证吗？**
微信扫码付款走的是微信支付的正常通道，你的微信账号本身绑了银行卡就能用，不需要在 DMIT 这边额外做实名认证。填写注册信息时邮箱要真实，因为 VPS 信息和验证邮件都发到那里去。

**Q：购买完多久能用？**
一般付款完成后几分钟到十几分钟内，邮件里就会有 IP 和登录方式了。DMIT 默认用 SSH 密钥登录（不是密码登录），第一次连接需要先配置好密钥对，官网有中文教程，新手照着做不复杂。

**Q：微信付款有没有手续费？**
没有，DMIT 不额外收付款手续费。你付多少就是多少，账单上显示的是美元价格，微信会按当天汇率自动换算成人民币扣款。

**Q：不满意能退款吗？**
购买后 3 天内，使用量不超过 30GB 的情况下可以申请全额退款。超出这个范围的退款要联系客服协商，官方有中文客服渠道。

**Q：香港 Premium 为什么贵那么多？**
物理距离是一方面——香港到国内延迟在 20-50ms，洛杉矶通常在 140-180ms。另一方面是线路质量：HKG.Pro 走电信 CN2 GIA、联通 AS9929、移动 CMI，这几条线路本身成本就高。如果你的用户主要在国内，或者跑的业务对延迟很敏感，这个溢价是有意义的；如果只是个人博客或者海外工具类项目，洛杉矶 T1 系列完全够用。
